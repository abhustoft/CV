## Current POC
* loopback
* React
* Ecmascript 2015
* Babel transpiler
* jspm module loader
* Otto VM

## Set up

* otto dev
* otto dev ssh
* make serve-src (Or: make build; make serve-dist)
* npm install
* cd server
* npm install
* jspm install
* node server.js
* In the browser, load `otto dev address`:3000
